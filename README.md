# Sentiment Analysis: DNN vs BERT

Proyek ini membandingkan performa model Deep Neural Network (TF-IDF + MLP) dan Transformer BERT dalam tugas klasifikasi sentimen menggunakan Amazon Polarity Dataset.

## Dataset
Amazon Polarity (Hugging Face)

## Models
- DNN (TF-IDF + MLP)
- BERT (bert-base-uncased)

## Evaluation Metrics
- Accuracy
- Macro F1-score

## Environment
- Python
- Google Colab
